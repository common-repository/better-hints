== Description ==

Adds custom notifications/hints for your visitors based on a condition to your site.

*NEW*

- new pulse effect
- more conditions (windows etc.)

- design your hints easy via the WordPress Customizer
- mobile-friendly and made with usability in mind
- works with WooCommerce
- for devs: add custom classes with code

Examples:

- add hint for android users
- add hint for a special page
- add hint for users in checkout

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the `Plugins` screen in WordPress
3. navigate to yoursite.com`/wp-admin/edit.php?post_type=hint` and create a hint
